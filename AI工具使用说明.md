## 1. 使用工具

Claude Code（全程，90%+ 代码由 AI 生成）

## 2. AI 完成的工作

页面设计（深色科技风格）、全部 HTML/CSS/JS 代码（6 个板块、项目详情弹窗、图片灯箱、滚动动画）、Git 推送部署

## 3. 手动修改

替换真实联系方式、筛选保留 4 个核心项目、调整板块顺序、准备 8 张项目截图、调大正文字体、创建 GitHub 仓库及 SSH 配置

## 4. 遇到的问题及解决

- 文字偏小 → AI 统一调大 6 处字体
- 项目卡片缺交互 → AI 添加详情弹窗（架构图+亮点列表+截图）
- 图片偶现加载失败 → 中文文件名 URL 编码 + 移除懒加载修复
- 国内网络无法推送 GitHub → 改用 SSH 密钥方式推送
- 板块顺序不合理 → 告诉 AI 后快速调整
